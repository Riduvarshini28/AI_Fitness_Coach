# 🏋️ AI Fitness Coach

An AI-powered personalized fitness coaching system that combines **Machine Learning** and **Rule-Based Expert Systems** to provide customized workout, diet, recovery, and health recommendations.

---

## 📌 Project Overview

This project predicts calories burned based on user fitness parameters and provides intelligent fitness recommendations using:

- Machine Learning models
- Expert system rules
- Fitbit fitness tracker dataset
- Flask web application
- Interactive data visualization

The system helps users receive personalized fitness guidance based on their:
- BMI
- Sleep hours
- Activity level
- Fitness goals
- Injury status
- Fatigue level
- Nutrition preference

---

## 🚀 Features

✅ Predicts daily calories burned  
✅ Personalized workout recommendations  
✅ Diet and nutrition suggestions  
✅ Sleep and recovery advice  
✅ Injury-aware fitness recommendations  
✅ Interactive Flask web interface

## 📊 Machine Learning Models

The project uses:

### Linear Regression
- Simple baseline prediction model
- Interpretable and efficient

### Random Forest Regressor
- Better accuracy
- Handles non-linear fitness data
- Improved calorie prediction performance
